# Retail Sales Data Analysis & Visualization Project

## Project Overview
This project focuses on creating a Retail Sales Data Agent, analyzing retail sales data, and building visualizations using Databricks.

The project is part of an assignment from BrightLearn and demonstrates the complete workflow of:
- Creating a Retail Sales Data Agent
- Preparing and analyzing retail sales data
- Generating business insights
- Creating visualizations and dashboards
- Documenting the full implementation process

---

## Objectives
The objectives of this project are to:

- Create a Retail Sales Data Agent in Databricks
- Analyze retail sales data
- Identify trends and patterns in sales performance
- Create data visualizations and dashboards
- Demonstrate practical data analytics skills
- Document the entire process step-by-step
